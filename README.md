## Cine Stream (Next.js)
A modern movie discovery web app built with Next.js 16, TypeScript, and Tailwind CSS.
Search movies, explore popular titles, and save your favorites  — all powered by the OMDB API.

---
## Features
- Search movies in real-time
- Browse popular movies
- Add/remove favorites (stored in localStorage)
- Fast performance with Next.js App Router
- Beautiful UI with Tailwind CSS
- Fully responsive design
---
## Usage
- Search Movies
1. Use the search bar in the navbar
2. Press Enter → redirects to /search?q=movie
- Favorites
1. Click the heart icon on any movie
2. View saved movies at /favorites
